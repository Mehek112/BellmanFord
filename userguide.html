User Guide for Bellman-Ford Simulation

Steps to Use the Simulator

  1. Add Edges:
    • Enter the source vertex, destination vertex, and edge weight in the respective input fields.
    • Click Add Edge to insert the edge into the graph.
    • Repeat the above steps for all the edges you want to add to the graph.
  2. Run Bellman-Ford Algorithm:
    • Enter the source vertex from where the algorithm will start.
    • Click Run to start the algorithm. The graph will start performing the Bellman-Ford algorithm.
  3. View Results:
    • The Shortest Distances table will display the minimum distances from the source vertex to all other vertices.
    • The Algorithm Log will show each step of the algorithm, including the edges relaxed and the distances updated.
  4. Reset the Graph:
    • Click Reset to clear the graph and all inputted edges.
